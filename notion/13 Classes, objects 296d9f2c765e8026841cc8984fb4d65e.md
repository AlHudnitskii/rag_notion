# 13. Classes, objects

Категория: Python Core
Статус: В процессе

### Что такое класс и объект?

**Класс** — это шаблон/чертёж для создания объектов. Определяет атрибуты (данные) и методы (функции).

**Объект** — это конкретный экземпляр класса с собственными данными.

![image.png](image%20106.png)

### Метод **init**

**`__init__`** — специальный метод (конструктор), вызывается автоматически при создании объекта.

### Параметр self

**`self`** — ссылка на текущий экземпляр класса. Через `self` обращаемся к атрибутам и методам объекта.

### Атрибуты класса vs атрибуты экземпляра

### Атрибуты класса

Общие для **всех экземпляров** класса. Определяются вне `__init__`.

![image.png](image%20107.png)

## Типы методов

### Методы экземпляра (Instance Methods)

Работают с конкретным экземпляром. Первый параметр — `self`.

### Методы класса (@classmethod)

Работают с **классом**, а не с экземпляром. Первый параметр — `cls`.

### Статические методы (@staticmethod)

Не работают ни с экземпляром, ни с классом. Обычные функции внутри класса.
