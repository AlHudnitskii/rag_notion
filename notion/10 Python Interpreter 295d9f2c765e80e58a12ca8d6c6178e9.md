# 10. Python Interpreter

Категория: Python Core
Статус: В процессе

Python Interpreter - это программа, которая читает и выполняет код на языке Python. Он преобразует человекочитаемый код в инструкции, понятные компьютеру, и исполняет их.

Python — это **интерпретируемый язык**, что означает:

- Код выполняется построчно
- Не требуется предварительная компиляция в машинный код (как в C/C++)
- Можно запускать код интерактивно

Основные реализации интерпретатора:

1. **CPython (стандартный)**
- **Самая популярная** реализация (написана на C)
- Официальная версия с python.org
- Преобразует код в байт-код, затем выполняет его
- Имеет GIL (Global Interpreter Lock)
- Поддерживает C-расширения
1. **Jython** 
- Написан на Java
- Компилирует Python в байт-код на Java
- Полная интеграция с Java-библиотеками
- Нет GIL
1. **IronPython**
- Написан на C# для .NET
- Интеграция с .NET библиотеками
- Работает на Common Language Runtime (CLR)

### Что такое байт-код?

- Промежуточное представление Python кода
- Независим от платформы (но зависит от версии Python)
- Хранится в `.pyc` файлах в папке `__pycache__`

## Python Virtual Machine (PVM)

**PVM** — это **stack-based виртуальная машина**, которая выполняет байт-код.
