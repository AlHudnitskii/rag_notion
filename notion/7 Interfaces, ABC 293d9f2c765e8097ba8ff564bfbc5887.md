# 7. Interfaces, ABC

Категория: Python Core
Статус: В процессе

Интерфейс - соглашение или набор правил, определяющих, как одна программная сущность взаимодействует с другой.

Работать с интерфейсами напрямую в Python нельзя, тем не менее можно использовать абстрактные классы. 

Абстрактный класс - класс, в котором созданы абстрактные методы - методы, которые обязательно должны присутствовать в дочерних классах. Создавать экземпляр абстрактного класса нельзя, его надо наследовать и уже у дочернего класса создавать экземпляр. В Python для этого используется модуль ABC (abstract base classes).  

![image.png](image%2046.png)

Методы в таком классе прописываются в помощью декоратора @abstactmethod.
